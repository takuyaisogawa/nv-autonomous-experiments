# Model Record Indexes

The manuscript comparison treats GPT-5.4, GPT-5.5, and GPT-5.6 Sol as parallel
benchmark conditions.  Each directory below provides the same entry points for
the Ramsey checkpoint and pODMR data evaluation records while preserving the
original source file locations.

| GPT-5.4 | GPT-5.5 | GPT-5.6 Sol |
| --- | --- | --- |
| [`gpt-5.4`](gpt-5.4/) | [`gpt-5.5`](gpt-5.5/) | [`gpt-5.6-sol`](gpt-5.6-sol/) |
| Ramsey and pODMR records | Ramsey and pODMR records | Ramsey and pODMR records |

Combined manuscript source tables are available in the [Ramsey](../ramsey/)
and [pODMR](../podmr/) directories.
